### Лекция 5: Объекты и основы DOM

```markdown
# Lecture 5 — Объекты и взаимодействие с DOM
**Тема:** Хранение данных в объектах и базовое управление HTML-элементами.
**Цель урока:** Понять структуру объектов и научиться менять содержимое страницы через JS.
**Время:** ~20 мин.
**Ключевые понятия:** Ключ-значение, DOM-дерево, querySelector, addEventListener.

## 1) Введение
До этого момента мы видели результаты только в консоли. DOM (Document Object Model) — это мост, который позволяет JS "видеть" теги HTML как объекты и менять их свойства в реальном времени.

## 2) Основная часть
### Объекты
Объекты группируют свойства сущности.
```javascript
const car = {
    brand: "Tesla",
    model: "S",
    drive: function() { console.log("Поехали!"); }
};
```
### Работа с DOM
Чтобы изменить страницу, нужно:
Найти элемент: document.querySelector('.class').
Изменить его: el.textContent = 'Новый текст'.
Подписаться на событие: el.onclick = ... или addEventListener
## 3) Пример/практика
Скрипт, меняющий цвет текста по нажатию кнопки:
```javascript
const btn = document.querySelector('#myBtn');
const title = document.querySelector('h1');

btn.addEventListener('click', () => {
    title.style.color = 'red';
    title.textContent = 'Кнопка нажата!';
});
```
