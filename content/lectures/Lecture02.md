## 📘 **Лекция 2 — Селекторы в CSS**

```markdown
# Lecture 2 — Селекторы в CSS

**Тема:** Способы выбора HTML-элементов для применения стилей: селекторы тегов, классов, идентификаторов и комбинированные.  
**Цель урока:** Научиться точно выбирать нужные элементы на странице с помощью различных типов селекторов.  
**Время:** ~20 мин.  
**Ключевые понятия:** селектор, селектор тега, селектор класса, селектор идентификатора, универсальный селектор, вложенность, каскадность.

---

### 1) Введение

Селекторы — это механизм, с помощью которого CSS "находит" HTML-элементы, чтобы применить к ним стили. Умение правильно выбирать элементы — ключевой навык в CSS. Чем точнее селектор, тем более предсказуемо работает стиль.

---

### 2) Основная часть

**Основные типы селекторов:**

| Тип селектора | Синтаксис | Пример | Описание |
|---------------|-----------|--------|----------|
| Селектор тега | `тег` | `p { }` | Выбирает все элементы с указанным тегом |
| Селектор класса | `.класс` | `.menu { }` | Выбирает элементы с атрибутом `class="класс"` |
| Селектор ID | `#id` | `#header { }` | Выбирает элемент с атрибутом `id="id"` (уникальный) |
| Универсальный | `*` | `* { }` | Выбирает все элементы на странице |

```css
/* Селектор тега — применяется ко всем <p> */
p {
    color: blue;
}

/* Селектор класса — применяется ко всем элементам с class="highlight" */
.highlight {
    background-color: yellow;
}

/* Селектор ID — применяется только к элементу с id="main-title" */
#main-title {
    font-size: 32px;
    color: red;
}

/* Универсальный селектор — применяется ко всем элементам */
* {
    margin: 0;
    padding: 0;
}
Комбинированные селекторы:
/* 1. Группировка — одинаковые стили для нескольких селекторов */
h1, h2, h3 {
    font-family: Arial, sans-serif;
}

/* 2. Вложенность (контекстные селекторы) — стили для элементов внутри других */
article p {
    line-height: 1.6;
    color: #333;
}

/* 3. Дочерний селектор (только прямые потомки) */
ul > li {
    list-style-type: square;
}

/* 4. Соседний селектор (следующий за элементом) */
h1 + p {
    margin-top: 0;
}

/* 5. Селектор атрибута */
input[type="text"] {
    border: 1px solid gray;
}

Пример HTML для практики:
<!DOCTYPE html>
<html>
<head>
    <style>
        /* Селектор тега */
        h1 { color: navy; }
        
        /* Селектор класса */
        .warning { color: red; font-weight: bold; }
        
        /* Селектор ID */
        #special { background-color: gold; }
        
        /* Вложенность */
        .container p { margin-left: 20px; }
        
        /* Группировка */
        .warning, .error { border: 1px solid red; }
    </style>
</head>
<body>
    <h1>Заголовок</h1>
    <p class="warning">Предупреждение!</p>
    <p id="special">Особый текст</p>
    <div class="container">
        <p>Текст внутри контейнера</p>
    </div>
</body>
</html>
```
### 3) Пример/практика
Задание:

Создайте HTML-страницу с:

Заголовком h1

Несколькими абзацами

Двумя кнопками (тег <button>)

Блоком <div class="card"> с текстом внутри

Напишите CSS, который:

Применяет к h1 синий цвет и выравнивание по центру

Применяет к абзацам размер шрифта 16px и отступ снизу 10px

Создает класс .btn для кнопок: зеленый фон, белый текст, padding 10px

Создает класс .btn-warning для второй кнопки: красный фон

Применяет к .card серую рамку, внутренние отступы 15px и скругленные углы 8px

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Селекторы CSS</title>
    <style>
        /* Ваш код здесь */
        
        /* Ответ: */
        h1 {
            color: blue;
            text-align: center;
        }
        
        p {
            font-size: 16px;
            margin-bottom: 10px;
        }
        
        .btn {
            background-color: green;
            color: white;
            padding: 10px;
            border: none;
            cursor: pointer;
        }
        
        .btn-warning {
            background-color: red;
        }
        
        .card {
            border: 1px solid gray;
            padding: 15px;
            border-radius: 8px;
        }
    </style>
</head>
<body>
    <h1>Учим CSS селекторы!</h1>
    <p>Первый абзац с информацией.</p>
    <p>Второй абзац для практики.</p>
    <button class="btn">Обычная кнопка</button>
    <button class="btn btn-warning">Опасная кнопка</button>
    <div class="card">
        <p>Это карточка с контентом</p>
    </div>
</body>
</html>
