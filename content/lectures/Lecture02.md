# Lecture 2 — Операторы и Управляющие конструкции
**Тема:** Математические и логические операторы, условия if/else.

## 1) Введение
Без условий программы были бы линейными и скучными. Логические конструкции позволяют скрипту принимать решения в зависимости от действий пользователя или состояния данных.

## 2) Основная часть
### Операторы сравнения
Важно помнить разницу между `==` (нестрогое) и `===` (строгое равенство). Всегда используйте `===`, так как оно проверяет и значение, и тип данных.

### Ветвление `if...else`
```javascript
if (age >= 18) {
    console.log("Доступ разрешен");
} else {
    console.log("Доступ запрещен");
}
```

### Логические операторы
&& (И) — возвращает true, если оба операнда верны.
|| (ИЛИ) — возвращает true, если хотя бы один верен.
! (НЕ) — инвертирует значение.

## 3) Пример/практика
### Проверка температуры для выбора одежды:
``` javascript
let temp = 25;
if (temp > 20 && temp < 30) {
    console.log("Надень футболку");
} else if (temp >= 30) {
    console.log("Сиди дома под кондеем");
}
```
