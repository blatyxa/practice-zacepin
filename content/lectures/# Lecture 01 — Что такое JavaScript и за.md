# Lecture 01 — Что такое JavaScript и зачем он нужен

**Тема:** Введение в JavaScript: история, возможности, применение  
**Цель урока:** Понять, что такое JavaScript, где и как он используется, и чем отличается от других языков  
**Время:** ~20 мин.  
**Ключевые понятия:** скриптовый язык, фронтенд, браузер, ECMAScript, динамическое программирование

### 1) Введение
JavaScript (JS) — это язык программирования, который делает веб-страницы живыми. Если HTML задаёт структуру, а CSS — внешний вид, то JavaScript отвечает за поведение: выпадающие меню, всплывающие окна, анимацию, отправку данных без перезагрузки страницы. Создан в 1995 году Бренданом Айком за 10 дней. Сегодня JS работает не только в браузере, но и на серверах (Node.js), в мобильных приложениях и даже на десктопах.

### 2) Основная часть
**Где выполняется JavaScript?** В браузере пользователя (клиентский JS), на сервере (серверный JS, например Node.js), в базах данных (MongoDB использует JS). **Что умеет JS в браузере:** изменять HTML и CSS, реагировать на действия пользователя (клики, ввод текста), отправлять запросы на сервер (AJAX, Fetch), устанавливать куки и работать с хранилищем, показывать уведомления и анимации. **Чего НЕ умеет JS в браузере:** читать/писать файлы на компьютере пользователя, получать доступ к другим вкладкам (без разрешения), напрямую управлять оборудованием.

### 3) Пример/практика
Откройте консоль разработчика в браузере (F12 → Console) и напишите:
```javascript
alert('Привет, мир!');
console.log('Это сообщение в консоли');
// Задание 1: Выведите в консоль своё имя и возраст
console.log('Имя: Иван');
console.log('Возраст: 25');

// Задание 2: Покажите всплывающее окно с текстом "JavaScript запущен!"
alert('JavaScript запущен!');

// Задание 3: Измените цвет фона любой страницы
// Откройте любой сайт и выполните в консоли:
document.body.style.backgroundColor = 'lightblue';
```
