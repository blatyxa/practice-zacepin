# Lecture 3 — Массивы и Циклы
**Тема:** Хранение коллекций данных и автоматизация повторяющихся действий.
**Цель урока:** Освоить методы работы с массивами и написание циклов for/while.
**Время:** ~20 мин.
**Ключевые понятия:** Index, push/pop, length, итерация, зацикливание.

## 1) Введение
В реальных приложениях мы работаем со списками: списки товаров, сообщений или пользователей. Массивы позволяют хранить их под одним именем, а циклы — обрабатывать их за миллисекунды.

## 2) Основная часть
### Массивы (Arrays)
Элементы в массиве нумеруются с нуля.
```javascript
let fruits = ["Яблоко", "Банан"];
fruits.push("Груша"); // Добавляет в конец
console.log(fruits[0]); // Яблоко
```
## Циклы
### Самый популярный — цикл for:
for (let i = 0; i < 5; i++) {
    console.log(`Шаг: ${i}`);
}
 ## Методы массивов
forEach — выполняет функцию для каждого элемента.
map — создает новый массив на основе старого.
## 3) Пример/практика
### Вывод всех элементов массива в консоль:
```javascript
const colors = ["Red", "Green", "Blue"];
for (let i = 0; i < colors.length; i++) {
    console.log(`Цвет №${i + 1}: ${colors[i]}`);
}
```
