# Lecture 4 — Функции: Основа логики
**Тема:** Объявление функций, аргументы и стрелочный синтаксис.
**Цель урока:** Научиться выносить повторяющийся код в функции и понимать область видимости.
**Время:** ~20 мин.
**Ключевые понятия:** Function Declaration, Arrow functions, return, scope.

## 1) Введение
Функция — это "черный ящик", который получает данные на вход, обрабатывает их и выдает результат. Это главный инструмент для декомпозиции кода.

## 2) Основная часть
### Способы объявления
1. **Function Declaration**: работает даже если вызвана выше объявления (благодаря всплытию).
2. **Arrow Function** (Стрелочная): краткий синтаксис, часто используется в колбэках.

```javascript
// Обычная
function sum(a, b) {
    return a + b;
}

// Стрелочная
const multiply = (a, b) => a * b;
``` 
### Область видимости (Scope)
Переменные, созданные внутри функции, не видны снаружи. Это защищает данные от случайного изменения.
## 3) Пример/практика
### Создание функции для расчета скидки:
```javascript
function getDiscountPrice(price, discount) {
    let finalPrice = price - (price * discount / 100);
    return finalPrice;
}
console.log(getDiscountPrice(1000, 15)); // 850
```
