# Lecture 1 — Введение в CSS и способы подключения

**Тема:** Знакомство с CSS, его роль в веб-разработке, способы подключения к HTML-странице и базовый синтаксис.  
**Цель урока:** Понять, зачем нужен CSS, научиться подключать его к HTML и применять базовые стили.  
**Время:** ~20 мин.  
**Ключевые понятия:** CSS, каскадные таблицы стилей, селектор, свойство, значение, тег `<style>`, внешний файл CSS, inline-стили.

---

### 1) Введение

CSS (Cascading Style Sheets) — это язык оформления веб-страниц. Если HTML отвечает за структуру и содержание (как скелет), то CSS — за внешний вид (как одежда и макияж). CSS позволяет задавать цвета, шрифты, отступы, позиционирование элементов и многое другое. Без CSS все веб-страницы выглядели бы как черно-белые текстовые документы.

---

### 2) Основная часть

**Способы подключения CSS к HTML:**

**1. Внешний файл (External) — РЕКОМЕНДУЕТСЯ**
Код CSS выносится в отдельный файл с расширением `.css` и подключается через тег `<link>`.

```html
<!-- index.html -->
<!DOCTYPE html>
<html>
<head>
    <title>Мой сайт</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Привет, мир!</h1>
</body>
</html>```

```css
/* styles.css */
h1 {
    color: blue;
    font-size: 24px;
} 
```
**2.Внутренний код (Internal)**
``` CSS
пишется внутри тега <style> в разделе <head> HTML-файла.
<!DOCTYPE html>
<html>
<head>
    <title>Мой сайт</title>
    <style>
        h1 {
            color: red;
            text-align: center;
        }
        p {
            font-size: 16px;
        }
    </style>
</head>
<body>
    <h1>Заголовок</h1>
    <p>Текст абзаца</p>
</body>
</html>
```
**3. Встроенный код (Inline)** 
CSS применяется непосредственно к HTML-элементу через атрибут style.
<h1 style="color: green; font-size: 30px;">Зеленый заголовок</h1>
 
Базовый синтаксис CSS:
``` селектор {
    свойство: значение;
    свойство: значение;
}
```
Селектор — указывает, к какому элементу применяются стили

Свойство — что именно меняем (цвет, размер, отступ)

Значение — как меняем (red, 16px, center)

``` 
/* Пример */
p {
    color: blue;           /* цвет текста */
    font-size: 18px;       /* размер шрифта */
    margin: 10px;          /* внешние отступы */
    background-color: yellow; /* фон */
}
```
<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <title>Моя первая CSS страница</title>
    <style>
        body {
            background-color: #f0f0f0;
            font-family: Arial, sans-serif;
        }
        h1 {
            color: #333;
            text-align: center;
            border-bottom: 2px solid #333;
            padding-bottom: 10px;
        }
        p {
            color: #666;
            line-height: 1.5;
            margin: 20px;
        }
        .highlight {
            background-color: yellow;
            font-weight: bold;
        }
    </style>
</head>
<body>
    <h1>Добро пожаловать на мой сайт!</h1>
    <p>Это обычный абзац текста.</p>
    <p class="highlight">Этот абзац выделен желтым фоном.</p>
</body>
</html>

### Задание:

Создайте HTML-файл и подключите к нему внешний CSS-файл style.css.

В CSS-файле задайте:

Для всей страницы (body) — цвет фона светло-серый

Для заголовка h1 — синий цвет и выравнивание по центру

Для абзацев — размер шрифта 16px и красный цвет

Создайте класс .important с жирным шрифтом и примените его к одному из абзацев.
