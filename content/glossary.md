# Глоссарий терминов JavaScript

**JavaScript** — язык программирования для создания интерактивных веб-страниц, выполняется в браузере пользователя.

**Переменная** — контейнер для хранения данных в программе, имеет имя и значение.

**let** — современный способ объявления переменной, значение которой можно изменять.

**const** — способ объявления константы, значение которой нельзя изменить после присвоения.

**var** — устаревший способ объявления переменной (использовался до появления let и const).

**Тип данных** — категория значения, определяющая какие операции можно с ним выполнять.

**Number (число)** — тип данных для целых чисел и чисел с плавающей точкой.

**String (строка)** — тип данных для текста, заключается в кавычки.

**Boolean (булевый)** — логический тип данных, принимает значения true (истина) или false (ложь).

**undefined** — тип данных, означающий что переменная объявлена, но не имеет значения.

**null** — специальное значение, означающее «ничего», «пусто» или «значение неизвестно».

**typeof** — оператор, возвращающий тип данных значения или переменной.

**Оператор** — символ или конструкция языка, выполняющая действие над данными.

**Арифметические операторы** — операторы для математических вычислений: +, -, *, /, %, **.

**Конкатенация** — операция соединения строк с помощью оператора +.

**Инкремент (++)** — оператор увеличения значения переменной на 1.

**Декремент (--)** — оператор уменьшения значения переменной на 1.

**Операторы сравнения** — операторы для сравнения значений: >, <, >=, <=, ==, ===, !=, !==.

**Строгое равенство (===)** — оператор сравнения, проверяющий равенство значений и их типов.

**Нестрогое равенство (==)** — оператор сравнения, проверяющий равенство значений с приведением типов.

**Логические операторы** — операторы для работы с булевыми значениями: && (И), || (ИЛИ), ! (НЕ).

**Условный оператор if** — конструкция для выполнения кода при выполнении условия.

**Условный оператор else** — часть условной конструкции, выполняющаяся если условие ложно.

**Тернарный оператор** — сокращенная запись if...else в виде: условие ? значение1 : значение2.

**switch** — конструкция для выбора одного из множества вариантов на основе значения.

**case** — ключевое слово в конструкции switch, обозначающее вариант для сравнения.

**break** — оператор прерывающий выполнение switch или цикла.

**default** — ключевое слово в switch, выполняющееся если ни один case не подошел.

**Цикл** — конструкция для многократного выполнения блока кода.

**for** — цикл с фиксированным количеством повторений (счетчиком).

**while** — цикл, выполняющийся пока условие истинно.

**do...while** — цикл, выполняющийся хотя бы один раз, затем проверяющий условие.

**Итерация** — одно выполнение тела цикла.

**Бесконечный цикл** — цикл, условие которого никогда не становится ложным (опасная ситуация).

**console.log()** — метод для вывода информации в консоль браузера.

**alert()** — функция, показывающая диалоговое окно с сообщением.

**Синтаксис** — набор правил написания кода на языке программирования.

**Комментарий** — текст в коде, который игнорируется при выполнении, используется для пояснений.

**Код** — текст программы, написанный на языке программирования.

**Функция** — блок кода, выполняющий определенную задачу (будет изучено позже).

**Массив** — структура данных для хранения списка значений (будет изучено позже).

**Объект** — структура данных для хранения коллекций свойств (будет изучено позже).

**Событие** — действие пользователя или браузера, на которое можно реагировать (будет изучено позже).

**DOM** — объектная модель документа, представляет HTML-страницу в виде объектов (будет изучено позже).
