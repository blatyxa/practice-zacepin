# Глоссарий терминов CSS

**CSS** — Cascading Style Sheets, язык стилей для оформления веб-страниц.

**Селектор** — шаблон для выбора HTML-элементов, к которым применяются стили.

**Свойство** — характеристика элемента, которую можно изменить (цвет, размер, отступ).

**Значение** — конкретная настройка свойства (red, 16px, center).

**Класс** — атрибут HTML для группировки элементов, в CSS обозначается точкой (.class).

**Идентификатор (ID)** — уникальный идентификатор элемента, в CSS обозначается решеткой (#id).

**Внешний CSS** — стили, вынесенные в отдельный файл с расширением .css.

**Внутренний CSS** — стили, записанные внутри тега <style> в HTML-документе.

**Inline-стили** — стили, применяемые непосредственно к элементу через атрибут style.

**Каскадность** — механизм определения приоритета стилей при конфликте правил.

**Наследование** — передача стилей от родительского элемента к дочерним.

**Блочная модель** — концепция, описывающая структуру элемента: content, padding, border, margin.

**Padding** — внутренний отступ между контентом и рамкой элемента.

**Margin** — внешний отступ от границы элемента до соседних элементов.

**Border** — рамка элемента.

**Box-sizing** — свойство, определяющее, включаются ли padding и border в ширину/высоту.

**Flexbox** — технология для создания гибких и адаптивных макетов.

**Flex-container** — родительский элемент с display: flex.

**Flex-item** — дочерний элемент внутри flex-контейнера.

**Justify-content** — свойство Flexbox для выравнивания по главной оси.

**Align-items** — свойство Flexbox для выравнивания по поперечной оси.

**Flex-direction** — свойство, задающее направление оси (row или column).

**Медиазапрос** — конструкция CSS для применения стилей в зависимости от характеристик устройства.

**Адаптивная верстка** — подход, при котором сайт корректно отображается на разных устройствах.

**Mobile First** — подход, при котором сначала разрабатывается версия для мобильных устройств.

**Шрифт** — гарнитура, определяющая внешний вид текста.

**Семейство шрифтов** — группа шрифтов с общими характеристиками (serif, sans-serif, monospace).

**Цветовая модель** — способ описания цвета (HEX, RGB, HSL).

**Градиент** — плавный переход между двумя или более цветами.

**Тень** — эффект, создающий объем у элемента (box-shadow, text-shadow).

**Скругленные углы** — свойство border-radius для создания закругленных углов элементов.

**Псевдокласс** — ключевое слово, добавляемое к селектору для задания состояния элемента (:hover, :active).

**Псевдоэлемент** — ключевое слово для стилизации части элемента (::before, ::after).

**Специфичность** — вес селектора, определяющий, какое правило будет применено.

**Позиционирование** — способ расположения элементов (static, relative, absolute, fixed).

**Float** — устаревший способ обтекания элементов текстом.

**Grid** — более новая технология для создания сложных двумерных макетов.

**Переменные CSS** — пользовательские свойства для хранения повторяющихся значений.
