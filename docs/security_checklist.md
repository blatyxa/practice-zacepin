### Какие шаги безопасности были выполнены.
*Изменение пароля админа*
После первого входа пароль был сразу же изменён на более сложный.
*Ограничение размера загрузки*
Было обновлено ограничение размера загрузки до 101 МБ, для большей оптимизации сайта.
*Проверка на права администратора*
Проводилась проверка для всех авторизованных пользователей на права администратора.
*Ограничение времени по которому пользователь находится на сайте бездействуя*
Было настроено ограничение времени (10 минут) по которому пользователь находится на сайте бездействуя, после истечения 10 минут пользователя выкидывает из сайта. Чтобы лучше улучшить оптимизацию сайта.
